## 按类别分类的扩展

所有扩展都位于 servers 文件夹中，每个扩展按 owner/repo 组织，并包含 `manifest.json` 和 `user_config.json`。如果 `manifest.json` 已通过社区验证，还会包含 `verified.json` 文件。
